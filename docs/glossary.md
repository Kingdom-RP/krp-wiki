# Глоссарий

Основные термины, которые встречаются в вики.

## Прокачка

**Путь развития**
: Группа связанных навыков (Война, Добыча, Промысел, Ремесло, Магия). Опыт всегда
  идёт в путь **целиком** — от действий любой его специализации.

**Специализация**
: Конкретный навык внутри пути (например, Кузнец в Ремесле). У каждой до **10
  уровней**. Прокачивается очками, которые даёт путь.

**Очко пути**
: Начисляется за новый уровень пути. Тратится на повышение уровня одной
  специализации этого пути.

**Опыт (XP)**
: Начисляется за действия навыка (добыча, крафт, варка, бой). Копится в путь,
  поднимая его уровень.

**Максимальный уровень пути**
: Сумма уровней всех его специализаций (например, 3 спец. × 10 = 30).

## Ограничения и доступ

**Гейт (ограничение по уровню)**
: Блокировка контента, пока навык не дорос. Может запрещать добычу блока, изъятие
  крафта, варку зелья, ношение снаряжения. Снимается с нужным уровнем.

**Лестница доступа**
: Список «уровень → что открывается». Показывает, с какого уровня специализации
  становится доступен ресурс, рецепт или действие.

**Гейт крафта / ношения**
: Один предмет может иметь **разные** требования на изготовление и на
  использование (напр. незеритовый меч: крафт — Кузнец 7, ношение — Воин 7).

## Крафт и качество

**Тир**
: Ступень ценности/сложности (материала, зелья, блюда, моба). Чем выше тир, тем
  выше нужный уровень и больше опыта.

**Закалка**
: Прочность **свежесделанного** инструмента/брони, зависящая от уровня. На уровне
  открытия ~⅓ прочности, на «открытие + 3» — полная.

**Дроп**
: Выпадение предмета из блока/моба.

**Двойной дроп**
: Шанс получить удвоенную добычу (руда, урожай, бревно).

**Партия сверх нормы**
: Шанс при крафте базовых заготовок получить бонусную пачку предметов.

## Прочее

**Ванильный**
: Стандартный, из базовой игры Minecraft (без модов).

**Модовый контент**
: Предметы/блоки из установленных модов. Обычно гейтятся **выше** ванильных.

**Dual-spec**
: Рецепт/действие, требующее уровней **двух** специализаций сразу.
